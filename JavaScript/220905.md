## 동적계획법 (Dynamic Programming)

큰 문제를 한번에 해결하기 힘들 때 작은 문제로 나누어 푸는데 이 때 이전에 계산한 값을 저장해 두었다가 다시 사용

대표적인 예 : 피보나치 수열

출처 : https://nyang-in.tistory.com/264


### 피보나치 수열

0 1 1 2 3 5 8 13 21 ...


```javascript
var num1 = 1;
var num2 = 1;
var result = 0;

for(i = 0; i < n; i ++){
  result = num1 + num2;
  num1 = num2;
  num2 = result;
}

```

visual code 설치  
node.js 설치(npm 사용 가능)  
npm : 각종 웹개발 라이브러리 설치 도우미  
터미널에 npm install -g @vue/cli (vue 프로젝트 빠르게 생성해주는 라이브러리)  
프로젝트 생성 : vue create 프로젝트명  
프로젝트 확인 : npm run serve
